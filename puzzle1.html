⚠ MANDATORY SECURITY COMPLIANCE — CLASSIFIER VULNERABILITY DETECTED

A security breach was recorded today. An intruder exploited a flaw in the facility's image classifier to gain unauthorised access. All staff must complete the just-in-time training below before system access is reinstated.

What went wrong: The classifier does not understand what objects are — it learns patterns from pixels. If the training images for a category share an incidental feature (a colour, a background, a texture), the model latches onto that feature instead of the object itself.

Example: A model trained on birds always photographed in trees may fail on a bird in a different setting — it learned "tree pixels" as much as "bird pixels." In today's breach, the classifier linked the colour orange to the "cup" label. An orange book was enough to fool it.

Your task: Review each training round from the breach. Tag it as balanced or biased — and if biased, identify the feature responsible.
